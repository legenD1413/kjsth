教程目录侧边栏组件示例

这个页面展示了如何使用教程目录侧边栏组件来显示教程、指南列表，保持网站风格的一致性。该侧边栏组件可以在网站的各个页面使用，方便用户快速查找所需的指南和教程。

如何使用

1. 在HTML中添加侧边栏容器

<div id="guides-sidebar"></div>

2. 在页面底部引入必要的CSS和JavaScript文件

<link rel="stylesheet" href="/assets/css/sidebar-guides.css">
<script src="/assets/js/guides-sidebar.js"></script>

3. 初始化侧边栏组件

// 创建指南侧边栏实例
const guidesSidebar = new GuidesSidebar('guides-sidebar', {
    title: '工具与指南',
    category: 'fba',
    viewAllUrl: '/guides/fba/',
    viewAllText: '查看全部'
});

组件参数

指南侧边栏组件支持以下配置选项：

  • title: 侧边栏标题
  • category: 指南类别
  • viewAllUrl: "查看全部"链接地址
  • viewAllText: "查看全部"按钮文本
  • guides: 指南数据数组（可选，如不提供则会从API获取）
  • apiUrl: 指南数据API地址
  • loadOnInit: 是否在初始化时加载数据

指南数据格式

指南数据应该符合以下格式：

[
  {
    "id": 1,
    "title": "指南标题",
    "description": "简短描述",
    "url": "/guides/path-to-guide.html",
    "icon": "fas fa-icon-name" // 可选的图标类
  },
  // 更多指南...
]
